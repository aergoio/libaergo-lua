# libaergo-lua

Library to interface with Aergo in Lua


## Installation

First [install libaergo](https://github.com/aergoio/libaergo#dependencies)

Then install this library:

```
git clone https://github.com/aergoio/libaergo-lua
cd libaergo-lua
make
sudo make install
```
